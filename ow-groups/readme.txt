Oasis Workflow Groups Add-on helps you create groups irrespective of roles and assign groups to workflow steps.

== Description ==
Oasis Workflow Groups Add-on is an efficient, and flexible solution  to create user groups and assign these groups to workflow steps.

The Groups Add-on has the following features:
1. Supports an unlimited number of groups.
2. Users can be assigned to any group irrespective of their role.
3. Users can be added/removed from the group at any time without affecting the workflow.
4.Once the groups are created, you can pre-assign groups to a specific step in the workflow.

== Installation ==
1. Download the plugin zip file to your desktop
2. Upload the plugin to WordPress
3. Activate your license by going to Workflows --> Settings --> License
4. Create Groups.
5. Assign groups to workflow steps.
